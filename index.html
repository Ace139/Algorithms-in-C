Repository where you can find the common algorithms in programming which are implemented in C

Compiler used : gcc-6.1.1 Operating System : Linux-based

The files with .c extensions are the C source code files, and the one with .o extension are the compiled output file.

Following is the list of Algorithms along with the name of Source Code file:

  • Linear Search (lsearch.c)
  • Binary Search using Recursion (bsearch.c)
  • Binary Search using Iterataion (bsearch2.c)
  • Selection Sort (selection.c)
  • Bubble Sort (bubble.c)
  • Insertion Sort (insertion.c)

Note : This repository and the list of algorithms will be updated continuously.
